>
> 1. 첫 협업 프로젝트 <br/>
> 2. 짧은 구현 기간 <br/>
> 3. 익숙한 어플리케이션​ <br/>

> **_서비스 주요 기능_**
>
> - 게시물 CRUD
> - 24시간 스토리 조회
> - 좋아요
> - 팔로우, 차단

#### Server List

- <i>gateway-server → API Gateway</i>
- <i>auth-server → 유저 인증 (로그인, 회원가입)</i>
- <i>feed-server → 게시물, 스토리, 댓글, 대댓글과 관련된 서비스 </i>
- <i>user-server → 유저 정보 수정</i>
- <i>graph-server → 유저간 관계 처리 ex. 팔로우, 차단</i>
- <i>alarm-server → 푸시(알림) 서비스</i>
